Fur-iend Finder

An application that connects potential pet owners with adoptable pets in shelters across the US. It initializes with a quiz of qualifications for the user.

The user is then shown an image of an adoptable animal, it's name, it's breed, it's age, and it's size. The user is prompted with an option to like or dislike an animal, saving it for later. A wikipedia page is linked to learn more breed information. The location of the shelter that currently houses the displayed pet is also displayed.

Tools used: PetFinder API, Wiki API, Google Maps API/OpenCage Data API, Animate.css, Local storage, Font Awesome
